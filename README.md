# 87calculators# 87 Calculator Engine

A high-performance, mobile-first utility hub featuring 87+ specialized calculators for mathematics, finance, health, and physics. Built for speed, precision, and a seamless user experience.

## Overview
87 Calculator provides instant, client-side calculations with a modern "Bloom" glassmorphism interface. Designed with a focus on core utility, the site offers zero-latency processing—meaning no server-side reloads, just pure speed.

## Key Features
* **87+ Specialized Tools**: Covers everything from complex financial EMI structures to health metrics and physics formulas.
* **Mobile-First Design**: Fully responsive UI built with Tailwind CSS, ensuring a premium experience on both desktop and mobile.
* **Instant Processing**: Calculations are handled entirely on the client side for maximum privacy and speed.
* **Accessibility**: Integrated Light/Dark mode with persistence via local storage.
* **SEO Optimized**: Includes essential metadata, sitemap, and robots.txt configuration for search engine discoverability.

## Technology Stack
* **Framework**: Astro
* **Styling**: Tailwind CSS
* **Deployment**: Vercel
